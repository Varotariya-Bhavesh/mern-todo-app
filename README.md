# 📝 MERN To-Do App

A simple and efficient To-Do List application built using the MERN stack (MongoDB, Express.js, React with Vite, and Node.js). This app allows users to manage daily tasks with basic CRUD operations.

---

## 🚀 Features

- ➕ Add new tasks
- 📄 View all tasks
- ❌ Delete tasks
- 🔄 Real-time updates
- 🌐 REST API integration

---

## 🛠️ Tech Stack

- Frontend: React (Vite), Axios
- Backend: Node.js, Express.js
- Database: MongoDB (Mongoose)
- Tools: Git, GitHub
